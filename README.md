# Axiom — Master Everything. In One Place.

Axiom is a premium, high-fidelity exam dashboard and AI-powered study companion designed for the modern Indian student. It replaces cluttered folders, messy WhatsApp groups, and broken Google Drive links with one unified, beautiful dashboard.

---

## ✨ Features

- 📂 **Resource Aggregator**: Drop in syllabus PDFs, textbooks, or class notes and organize them instantly.
- 🤖 **Context-Aware AI Study Buddy**: An AI companion that answers queries, resolves doubts at 2 AM, and generates summaries using *your uploaded materials* instead of generic web search.
- 📝 **PYQ & Question Banks**: Find past year questions and generate solutions aligned with specific marking schemes.
- ⚡ **Smart Revision Modules**: Instantly turn chapters into revision summaries, flashcards, or quick-fire quizzes before exam night.
- 🏆 **Gamified Progress Tracking**: Built-in trackers for study streaks, target goals, performance analytics, and XP rewards.

---

## 🎨 Tech Stack & Design System

Axiom is built from the ground up with clean, modern web technologies:

- **HTML5**: Structured semantic markup optimized for accessibility.
- **Vanilla CSS**: Premium dark-mode themes, glassmorphism overlays, animated gradients, and 3D device visual cues (like the interactive 3D hero device wrapper).
- **JavaScript (ES6+)**: Custom dynamic states, interactive features, scroll-progress bars, and accordion/carousel controls.

---

## 🚀 SEO & Integration Features

Axiom includes complete search engine optimization assets out of the box:
- **Metatags & OG tags**: High-fidelity Open Graph and Twitter Card tags for visual link-sharing on WhatsApp, Twitter/X, and LinkedIn.
- **JSON-LD Schema**: Configured `SoftwareApplication` / `EducationalApplication` structured data.
- **Canonical Routing**: Canonical tags defined.
- **Crawlers Configuration**: Complete `robots.txt` and `sitemap.xml` templates in the root folder.

## 🛠️ Getting Started

To run Axiom locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Axiom.git
   ```
2. Open `index.html` directly in a browser, or run a simple local web server:
   ```bash
   # Using Node.js (npx)
   npx serve .

   # Or using Python
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` or `http://localhost:3000` in your web browser.
